# TKUITOCC 淡江大學資訊科技及開放文化社群

## 關於我們

淡江大學資訊科技及開放文化社群致力於推動淡江大學校內外的資訊科技及開放文化發展，並與其他社群互相合作、交流、學習。

資訊科技方面包括專案開發（行動應用程式開發、機器人開發等）、資料結構化、程式語言學習與經驗交流等。

開放文化則包含數位共筆（wiki）、開放原始碼、開放資料、數位典藏、多語言翻譯、譯名標準化等。

## 目前服務

* 開放資料
* 資訊科技交流
* 專案開發
* 社群治理
